# AmazingPuzzle 主要用于将多张图片拼接成单张图片

- 支持1~5张图片
- 每张单独的照片支持移动缩放
- 每组图片有6中风格类型
- 纯Swift 语言开发, 简单集成使用

# CocoaPods

```
  pod 'AmazingPuzzle'
```

# Requirements
* iOS 9.0+
* Swift 5.x
* Xcode 12.x
